# Electron Desktop Application Requirements

# ==============================================
# Node.js & npm Version Requirements
# ==============================================
# Node.js: v14.x or higher (v18.x recommended)
# npm: v6.x or higher (v9.x recommended)

# ==============================================
# Core Dependencies
# ==============================================
# These dependencies are listed in package.json and will be installed via npm

# - None (standalone Electron app)

javascript-obfuscator@^4.1.1        # JavaScript obfuscation for production

# ==============================================

# - Windows 7 or later

# - macOS 10.13 or later

# For Linux builds:
# - Ubuntu 18.04 or later (or equivalent)
# - Build essentials: build-essential, libssl-dev

# ==============================================
# Installation Instructions
# ==============================================

# 1. Install Node.js and npm:
#    Download from: https://nodejs.org/

# 2. Clone the repository:
#    git clone <repository-url>
#    cd traffic-simulator-desktop

# 3. Install dependencies:
#    npm install

# 4. Run in development mode:
#    npm start

# 5. Build for your platform:
#    npm run build:win    (Windows)
#    npm run build:mac    (macOS)
#    npm run build:linux  (Linux)

# ==============================================
# Optional Tools
# ==============================================
# Git - for version control
# Visual Studio Code - recommended IDE
# Chrome DevTools - for debugging (built into Electron)

# ==============================================
# Notes
# ==============================================
# - This is an Electron-based application, not a Python project
# - The main application is a standalone HTML file with embedded CSS and JavaScript
# - All JavaScript dependencies are either bundled or loaded from CDN
# - No server-side components required
# - Cross-platform compatible (Windows, macOS, Linux)
